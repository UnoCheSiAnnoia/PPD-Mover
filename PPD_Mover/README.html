PPD Mover

What is this

PPD Mover is a program that should help putting songs into PPD.
PPD is an emulator for the Project DIVA game.

I am a beginner at programming so I don't know if this will come out well in the end, but I am working on this project and I am trying to make it as good as I can with my skills

How To Use

NOTE: you can drag a file in every entry if you don't want to manually select every file

  1. Press the "select" button and select the "songs" folder. To find it, search for the folder of PPD, the songs folder should be in the same place of the executable
  2. If you want to save the folder you selected for future use, you can press the "save folder" button, if you don't want to, go to the next step
  3. Now you can press the "Change directory" button to change the starting directory for selecting the .zip file containing the song, if you don't want to, go to the next step
  4. Press "Browse files" to search for a .zip file
  5. Press "Move" to unzip and move the file you selected to the songs folder
  6. If you want to change the starting folder for the video selection, you can press the "Change directory" button, if you don't want to, go to the next step
  7. Press the "Select" button to select the video to move (only mp4 are allowed)
  8. Press the "Select folder" button and select the folder for the song you have extracted (the starting folder is the one you selected in the "select directory" part)
  9. Press "Move video" to move the video selected to the folder you just selected

BONUS:

  1. You can press "delete saves" to delete all the saves (I know, it wasn't obvious)
  2. You can check the "Delete .zip" checkbox to delete the zip file when you move it

If you find a bug in the app, please report it on the issue page on github so hopefully I will be able to fix it
